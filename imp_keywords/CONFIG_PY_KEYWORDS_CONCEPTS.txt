================================================================================

CONFIG.PY CONCEPTS EXPLAINED
================================================================================

A. BaseSettings from Pydantic
- Use to load settings from environment variables and .env files.
- Great for simple configuration management in Python.

B. Environment Variables
- Keep secrets and deployment settings out of source code.
- Use os.getenv() to read values safely.

C. DATABASE_URL Fixing
- Convert old "postgres://" URLs to "postgresql://".
- Append "+psycopg2" so SQLAlchemy uses the correct DB driver.
- Strip quotes and spaces that may appear when copying from dashboards.

D. Field Validator
- Runs before the setting value is parsed.
- Helps normalize and validate configuration input.

E. JWT Configuration
- SECRET_KEY signs tokens securely.
- ALGORITHM defines the JWT hashing method.
- ACCESS_TOKEN_EXPIRE_MINUTES controls token lifetime.

F. App Metadata
- APP_NAME and APP_VERSION let the app identify itself.
- Used in docs, logs, and version checks.
